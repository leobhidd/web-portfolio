# Portfolio técnico

Proyecto personal desarrollado para demostrar habilidades prácticas en desarrollo de software, enfocado en backend, APIs y Automatización

## Objetivo del proyecto

Este proyectoi funciona como curriculum online y portafolio técnico.
Con el objetivo de demostrar experiencia real en:

-Backend con Python (FastAPI).
-Desarrollo de APIs Rest.
-Automatización y Web Scrapping.
-Buenas prácticas de desarrollo.
-Deploy y control de versiones.

## Stack tenológico

- **Lenguaje:** Python
- **Backend:** FastAPI
- **Base de Datos:** PostgreSQL
- **Frontend:** HTML, CSS, JavaScript (React en Progreso)
- **DevOps:** Docker (Planificado)
- **Versionamiento:** Git + GitHub
